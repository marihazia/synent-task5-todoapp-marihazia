# TaskFlow - To-Do List Web App

A task management web application built as part of the Synent Technologies Web Development Internship Program.

## Task
**Task 5 - Intermediate Level**

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks by All / Active / Completed
- Tasks persist after page refresh using localStorage
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- localStorage API

## How to Run
1. Clone the repository
2. Open `index.html` in any browser
3. No setup or installation required
